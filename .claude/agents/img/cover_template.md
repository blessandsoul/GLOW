## Base Concept
A seamless abstract pattern or texture filling the entire frame. No text, no titles, no letters, no words, no objects. Pure visual pattern inspired by the gemstone's color and material essence. Each pack gets a unique organic or geometric pattern driven by the stone's natural texture.

## Base Prompt Structure

```
A seamless [PATTERN_TYPE] filling the entire frame, [PATTERN_DETAIL], color palette built from [STONE_PALETTE], [TEXTURE_QUALITY], [DEPTH_EFFECT], [LIGHTING], [MOOD_WORD] atmosphere, no text no letters no words no titles no watermarks, abstract pattern design, 1:1 square format.
```
