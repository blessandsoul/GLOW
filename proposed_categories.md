# Proposed Beauty Service Categories

This document contains the corrected and highly detailed Georgian terms for beauty salon services. Missing popular items have been added to ensure the list is comprehensive for masters.

## 1. ლეში ინდუსტრია / წამწამები (Lashes)
- წამწამების დაგრძელება (კლასიკური)
- წამწამების მოცულობითი დაგრძელება (2D, 3D, Volume)
- წამწამების კორექცია
- წამწამების მოხსნა
- წამწამების ლამინირება
- წამწამების ბიოდახვევა (Lash Lift)
- წამწამების შეღებვა

## 2. წარბები (Brows)
- წარბების არქიტექტურა და ფორმირება
- წარბების ლამინირება
- წარბების შეღებვა (საღებავით / ხნით)
- წარბების კორექცია ძაფით
- წარბების კორექცია ცვილით
- წარბების კორექცია პინცეტით
- მიკრობლეიდინგი (Microblading)
- წარბის პუდროვანი დაფერვა (Powder Brows)

## 3. ფრჩხილები (Nails)
- კლასიკური მანიკური
- აპარატული / კომბინირებული მანიკური
- კლასიკური პედიკური
- აპარატული პედიკური
- შილაქი / გელ-ლაქით გადაფარვა
- შილაქის მოხსნა
- ფრჩხილის დაგრძელება (გელით / პოლიგელით / აკრილით)
- ფრჩხილის გამაგრება (ბაზით / გელით)
- ფრჩხილის დიზაინი
- ფრენჩი / ომბრე (French / Ombre)
- მამაკაცის მანიკური / პედიკური

## 4. მაკიაჟი (Makeup)
- დღიური მაკიაჟი
- საღამოს / სადღესასწაულო მაკიაჟი
- სარძლო მაკიაჟი
- მაკიაჟი ფოტოსესიისთვის
- ტუჩის პერმანენტული მაკიაჟი
- წარბის პერმანენტული მაკიაჟი
- ქუთუთოს პერმანენტული მაკიაჟი (ისარი / კონტური)

## 5. თმა (Hair)
- ქალის თმის შეჭრა
- მამაკაცის თმის შეჭრა
- ბავშვის თმის შეჭრა
- თმის დავარცხნა / სადღესასწაულო ვარცხნილობა
- თმის შეღებვა (სრული / ფესვები)
- მელირება (ჰაილაითინგი)
- ბალაიაჟი / ომბრე / შატუში
- რთული ტექნიკით შეღებვა
- თმის ტონირება
- თმის ქიმიური / ბიო დახვევა
- თმის დაგრძელება
- კერატინით გასწორება
- თმის ბოტოქსი
- აფრო ნაწნავები (Braids)

## 6. კანის მოვლა (Skincare)
- სახის წმენდა (მექანიკური / ულტრაბგერითი)
- სახის პილინგი
- სახის სკულპტურული მასაჟი
- კარბოქსითერაპია
- აპარატული კოსმეტოლოგია (Hydrafacial, RF-ლიფტინგი)
- მიკრონიდლინგი
- ბიორევიტალიზაცია
- მეზოთერაპია
- კონტურული პლასტიკა (ფილერი)
- ბოტულინოთერაპია (ბოტოქსი)
- პლაზმოთერაპია (PRP)

## 7. ეპილაცია და რუჯი (Waxing, Laser & Tan)
- ლაზერული ეპილაცია
- ელექტრო ეპილაცია
- დეპილაცია ცვილით
- შუგარინგი
- სახის / ტუჩის დეპილაცია ძაფით
- სოლარიუმი
- ავტორუჯი (Spray tan)

## 8. მასაჟი და სხეული (Body)
- სარელაქსაციო მასაჟი
- სამკურნალო / სპორტული მასაჟი
- ანტიცელულიტური მასაჟი
- ლიმფოდრენაჟული მასაჟი
- სხეულის შეფუთვა / პილინგი
- ენდოსფერო თერაპია
- პრესოთერაპია

## 9. ცხოვრების სტილი და სხვა (Other)
- ყურის / ცხვირის გახვრეტა (Piercing)
- ტატუირება
- ტატუს ლაზერული მოშორება
- სტილისტის / შოპერის კონსულტაცია
