# Glow.GE — Рекламная кампания

## Продукт
**Glow.GE** — AI-платформа для бьюти-мастеров (Грузия).
Улучшение фото, портфолио, брендинг, подписи для инсты — всё в одном месте.

## Целевая аудитория
- Мастера по ресницам, визажисты, мастера маникюра, парикмахеры
- Грузия (расширение: Кавказ → Европа)
- Языки: RU (основной) + GEO (перевод)

## Ключевые боли ЦА
1. **Фото с телефона** выглядят непрофессионально → теряются клиенты
2. **Фотограф дорого** — 200-400₾ за сессию каждый месяц
3. **Фотошоп долго** — 2+ часа на одну фотку
4. **Нет портфолио** — клиенты не могут оценить качество
5. **Нет бренда** — все мастера выглядят одинаково в инсте

## Ключевое предложение
- AI улучшает фото за 2 минуты (2 варианта)
- Портфолио + брендинг + подписи = всё в одном
- **Бесплатно** — 3 фото/месяц, без карты, без обязательств
